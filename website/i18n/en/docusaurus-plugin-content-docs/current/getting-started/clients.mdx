# HTTP clients

To interact with the Facturapi API, you can use any HTTP client that supports REST requests.

Here are some resources to get started testing Facturapi with popular HTTP clients:

### Postman

You can import the Facturapi Postman collection from the following link:

[Postman collection](https://www.postman.com/satellite-technologist-58292776/facturapi/collection/dszb8xi/facturapi)

### OpenAPI

You can also download the Facturapi OpenAPI file to use in your favorite HTTP client:

[OpenAPI spec](https://docs.facturapi.io/redocusaurus/api-en.yaml)
