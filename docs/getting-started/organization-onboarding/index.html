Configuración inicial de una organización

Esta guía resume el checklist de configuración en dashboard para dejar tu organización lista para emitir CFDI.

Checklist de configuración inicial​

1. Crear la organización​

Al crear tu cuenta e iniciar el proceso de configuración inicial en dashboard, la primera pantalla te pedirá el nombre de tu organización. Captura ese nombre y continúa para crear la organización emisora.

Si trabajas con varias organizaciones (multi-RFC), consulta también Organizaciones (Multi-RFC). Si prefieres crear organizaciones por API, usa Crear Organización.

2. Subir certificados CSD​

En el dashboard ve a Configuración > Certificados y carga:

  • Archivo .cer
  • Archivo .key
  • Contraseña de la llave privada

Sin CSD no podrás timbrar CFDI en ambiente live.

3. Completar datos fiscales​

En Configuración > Datos fiscales captura los datos legales/fiscales de la organización (RFC, razón social, régimen, código postal, etc.).

4. Activar suscripción​

En Cuenta > Suscripción activa un plan para la organización.

Este paso forma parte de la configuración base para operar en producción con la organización.

5. Firmar Carta Manifiesto (Requisito SAT)​

En Configuración > Firma de Carta Manifiesto firma la carta para cumplir con la RMF vigente.

Si eres integrador con varias organizaciones emisoras, puedes compartir la url de nuestro portal público para firmar la carta manifiesto: https://www.facturapi.io/manifiesto

En este paso se usa la e.firma (FIEL) del RFC emisor para firmar el manifiesto. Para completar la firma, se solicita:

  • Archivo .cer de e.firma
  • Archivo .key de e.firma
  • Contraseña de la llave privada

Al finalizar, podrás descargar los archivos de la Carta Manifiesto en formatos PDF, XML o ZIP.

Paso opcional recomendado​

Personalización de facturas​

En Configuración > Personalización puedes cargar logotipo, colores y preferencias de render para el PDF.

No es obligatorio para completar la configuración fiscal base, pero sí recomendable para estandarizar la salida de tus comprobantes.

Verificación rápida​

Antes de emitir en live, valida que tu organización tenga:

  • Organización creada
  • CSD cargado
  • Datos fiscales completos
  • Suscripción activa
  • Carta manifiesto firmada
