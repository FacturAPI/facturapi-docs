Complements

Currently, FacturAPI supports the issuance of all types of CFDI as well as all SAT add-ons or complements. You can see how to create different types of CFDI documents in the Create Invoice and Create Retention references.

note

You can insert any complement by adding the corresponding XML in the "complements" node of your CFDI.

tip

When inserting a complement, we will add it to the XML at the time of stamping; since the SAT does not require that the information of all the complements be printed on the PDF, it will only appear in the XML. If you need certain information from the complement to be added to the PDF, you can use the pdf_custom_section field.
