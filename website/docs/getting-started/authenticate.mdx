# Autenticación

Para poder llamar a cualquier método de la API, necesitarás identificarte
usando una de las 3 [llaves secretas](/docs/getting-started/concepts/#llaves-secretas).

Obtenen tus llave secretas creando una cuenta gratuita en
[Facturapi](https://www.facturapi.io/register?utm_source=facturapi-docs&utm_medium=GithubPages).

## Usando una de nuestras librerías

Usa la llave secreta al momento de crear la instancia del cliiente

<Tabs groupId="codeExamples">
<TabItem value="js" label="Node.js" default>

```javascript
const Facturapi = require('facturapi');
const facturapi = new Facturapi('<SECRET_KEY>');
```

</TabItem>
<TabItem value="cs" label="C#">

```csharp
var facturapi = new FacturapiClient("<SECRET_KEY>");
```

</TabItem>
<TabItem value="php" label="PHP">

```php
$facturapi = new Facturapi( "<SECRET_KEY>" );
```

</TabItem>
</Tabs>

## Usando tu propio cliente HTTP

Facturapi utiliza el método de autenticación HTTP
[Bearer Token](https://swagger.io/docs/specification/authentication/bearer-authentication/).

El token a utilizar es una de las 3 **Llaves Secretas**, explicadas con detalle
en la sección de [llaves secretas](/docs/getting-started/concepts/#llaves-secretas).


<Tabs groupId="codeExamples">
<TabItem value="curl" label="cURL">

Cada petición deberá incluir tu llave secreta en el
encabezado de autorización.

```bash
curl https://www.facturapi.io/v1/<endpoint_method> \
  -H "Authorization: Bearer <SECRET_KEY>" \
```

</TabItem>
</Tabs>
