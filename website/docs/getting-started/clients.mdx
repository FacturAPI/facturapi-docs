# Clientes HTTP

Para interactuar con la API de Facturapi, puedes utilizar cualquier cliente HTTP que soporte solicitudes REST.

Estos son algunos recursos para empezar a probar Facturapi con clientes HTTP populares:

### Postman

Puedes importar la colección de Postman para Facturapi desde el siguiente enlace:

[Postman collection](https://www.postman.com/satellite-technologist-58292776/facturapi/collection/dszb8xi/facturapi)

### OpenAPI

También puedes descargar el archivo OpenAPI de Facturapi para usarlo en tu cliente HTTP favorito:

[OpenAPI spec](https://docs.facturapi.io/redocusaurus/api-es.yaml)
