# Recibo de Nómina

Un comprobante de tipo Nómina, también llamado **recibo de nómina**, se utiliza para dar constancia que avale el pago ante la legislación laboral.
Este tipo de CFDI debe emitirse por cada pago realizado por el concepto de remuneraciones de sueldos, salarios y asimilados a estos y su objetivo es formalizar la relación entre empleador y empleado, además de ser un requisito para cumplir con las normativas fiscales impuestas por la ley.
:::info Los recibos de nómina también sirven para deducción de impuestos de una empresa
Con ellos se comprueban aportaciones patronales realizadas al Instituto Mexicano del Seguro social (IMSS), Instituto del Fondo Nacional de la Vivienda para los Trabajadores (INFONAVIT) así como el pago del impuesto sobre la nómina.
:::

El CFDI de nómina puede identificarse como tipo de comprobante "N"
